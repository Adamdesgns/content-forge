# Content Forge — Web / PWA build

Generated by `npm run build:web`. This folder is a self-contained static site.

## Test on your phone (same Wi-Fi)
```bash
cd web
npx --yes serve .        # or: python3 -m http.server 8080
```
Open the printed `http://<your-computer-ip>:PORT` URL on your phone.
In Safari (iOS): Share → **Add to Home Screen**.
In Chrome (Android): menu → **Install app** / **Add to Home screen**.

## Deploy (public HTTPS — required for install + offline)
Drag this `web` folder onto **Netlify Drop** (app.netlify.com/drop), or use
Vercel / GitHub Pages / Cloudflare Pages. Any static host works.

Data (projects, brand defaults) is stored in the browser via localStorage.
The full prompt library and app shell are cached for offline use.
